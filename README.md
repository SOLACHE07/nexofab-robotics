# NexoFab Robotics

Proyecto de práctica académica: web industrial responsive publicada con GitHub Pages.

## Empresa ficticia
NexoFab Robotics es una empresa inventada dedicada a automatización industrial, robótica colaborativa, PLC, SCADA e IoT industrial.

## Archivos principales
- `index.html`: estructura principal de la web.
- `style.css`: estilos personalizados.
- `script.js`: interacción JavaScript.
- `img/`: imágenes ficticias generadas para la práctica.
- `capturas/`: capturas de escritorio y móvil.
- `memoria_practica_NexoFab_Robotics.docx`: breve memoria de entrega.

## Secciones de la web

7. Pie de página académico

## Publicar en GitHub Pages
1. Crear un repositorio nuevo en GitHub.
2. Subir los archivos de esta carpeta manteniendo `index.html` en la raíz.
3. Entrar en **Settings > Pages**.
4. Seleccionar la rama `main` y guardar.
5. Copiar el enlace público generado por GitHub Pages.
